What does it do?

This is an application for predicting mpg (Miles/ Gallon). It takes predictor values as chosen by user and fits those values in a linear data model. The fitted model is then utilized for predicting the mpg value.

Operational Instructions

To use this application -

  • Please choose feature variables using the tickboxes at the very left side of the page.
  • Use the control widgets to pick a value for features selected in step 1.
  • The green area will display the outputs based on your choice.
  • Please explore the blue area to know more about the internal mechanisms of the application.

Key Details ( Under the hood )

  • The application uses R's mtcars dataset as training set for fitting the model. Hence the input ranges are taken from the mtcars dataset.
  • It dynamically builds the formulae for fitting into a linear model everytime user changes feature selection.
  • Index.html is used instead of ui.R for ease of user interface design.
